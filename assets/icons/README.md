# Icon Library

リファレンス画像で使われるようなフラットアイコンの想定マッピング。
現状の実装ではNano Bananaに絵文字を渡して「このスタイルのアイコンを描いて」と指示しているが、
将来的に PIL テンプレート方式に切り替える際の参考とする。

| 絵文字 | テーマ | 推奨色 |
|---|---|---|
| 🛢 | 原油 / コスト / 原料 | gray |
| ⏰ | 時間 / タイミング | red |
| 📦 | 商品 / 日用品 | brown |
| 📈 | 値上げ / 増加 | green |
| ⚠️ | 警告 / リスク | orange |
| 💰 | お金 / 節約 | gold |
| 📊 | データ / 統計 | blue |
| 🔑 | ポイント / 要点 | purple |
| ✅ | チェック / 完了 | green |
| 💡 | アイデア / TIPS | yellow |
| 🏠 | 家 / 暮らし | brown |
| 🔋 | 電気 / エネルギー | green |
| 🌧 | 天気 | blue |
| 🛒 | 買い物 | red |
| 📱 | アプリ / デジタル | blue |

新しいアイコンが必要になったら、generate_tree.py の SYSTEM_PROMPT の絵文字リストを増やす。
